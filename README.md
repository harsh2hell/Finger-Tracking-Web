# Finger Tracking Web App

A real-time finger tracking web application built using HTML, CSS, and JavaScript.
The project uses browser-based computer vision to detect and track fingers through a webcam.

## Features
- Real-time hand and finger tracking
- Runs directly in the browser
- No installation required
- Single-file lightweight implementation

## Tech Stack
- HTML5
- JavaScript
- MediaPipe Hands (via CDN)
- Web Camera API

## How to Run
1. Download or clone this repository
2. Open `index.html` in a modern browser (Chrome / Edge recommended)
3. Allow camera access

## Notes
This project was created using AI-assisted development (Gemini) and further tested and organized for learning and demonstration purposes.

## Applications
- Gesture-based interaction
- Computer vision learning
- Web-based HCI experiments
